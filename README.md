# Customer Purchase Behavior Analysis

This project analyzes customer purchase behavior using Python and Pandas.

The goal is to uncover insights about:
- Customer spending patterns
- Best-selling products
- Revenue trends over time
- High-value customers
- Customer segmentation using RFM analysis

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights

- A small number of customers generate a large portion of total revenue.
- Certain products dominate overall sales volume.
- Monthly sales trends show seasonal spikes.
- RFM analysis identifies high-value and loyal customer segments.

## Dataset

Online Retail dataset from the UCI Machine Learning Repository.
